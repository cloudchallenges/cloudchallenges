## What is Cloud Challenges?

Cloud Challenges provides practical problem statements designed to build real cloud skills. Each challenge includes:

- **Clear scenario** ‚Äî Real-world context and requirements
- **Architecture guidance** ‚Äî Visual diagrams and component breakdowns
- **Starter code** ‚Äî Boilerplate to get you started quickly
- **Solution walkthroughs** ‚Äî Detailed explanations when you need help

## Why?

Most cloud tutorials walk you through step-by-step. Cloud Challenges takes a different approach ‚Äî you're given a problem to solve, not a script to follow. This builds the problem-solving skills that matter in real projects.

## Getting Started

1. Browse challenges at [cloudchallenges.hyperoot.dev](https://cloudchallenges.hyperoot.dev)
2. Pick a challenge that matches your skill level
3. Clone the starter repository
4. Build your solution
5. Compare with the provided solution guide
